The code is compiled with `pyinstaller pdfmerge.py`. 
Add '\pypdf\dist\pdfmerge' to path. 
Run 'pdfmerge file1.pdf file2.pdf' from a command prompt/PowerShell, and the merged pdf is output to 'merged.pdf'.
